# Real Dashboard Mirror V1.0 Free Beta

Public beta download for Real Dashboard Mirror / DashMirror.

## Downloads

- Windows single-file EXE: `DashMirrorOverscan-V1.0-FreeBeta-Exe.zip`
- Windows portable fallback: `DashMirrorOverscan-V1.0-FreeBeta-PortableFallback.zip`
- Android Viewer APK: `DashMirrorViewer-V1.0-FreeBeta-Android-debug.apk`
- Android unsigned release APK: `DashMirrorViewer-V1.0-FreeBeta-Android-release-unsigned.apk`

## What It Does

- Captures a Windows game/window using Windows Graphics Capture.
- Places the source window with a bottom overscan area for hidden dashboard content.
- Streams the dashboard crop over the local network.
- Provides Android and browser viewers.
- Provides a TV browser endpoint at `http://<pc-lan-ip>:5055/tv?fps=60&w=1280`.
- TV mode includes a `Phone mode` button to return to the normal viewer.

## Free Beta Mode

Free mode allows 10 minutes per capture session. A beta license unlock can enable unlimited capture during the beta period.

## Quick Start

1. Download and unzip `DashMirrorOverscan-V1.0-FreeBeta-Exe.zip`.
2. Double-click `DashMirrorOverscan.exe`.
3. Launch your racing game in windowed or borderless mode.
4. Select the game source in DashMirror.
5. Apply a bottom overscan preset.
6. Refresh the game resolution or placement.
7. Confirm the dashboard appears in the bottom overscan preview.
8. Adjust and validate the dashboard crop.
9. Start the viewer.
10. Scan the QR code or open the shown LAN URL on Android or TV, for example `http://192.168.x.x:5055/`.

The QR code contains the plain LAN URL, not an Android-only `intent://` link, so it works with phones, tablets, TV browsers, and generic QR scanners.

## Android Viewer

Install `DashMirrorViewer-V1.0-FreeBeta-Android-debug.apk` by sideloading it on your Android device. The APK is a LAN viewer only; it does not capture the game, run telemetry, use WebRTC, or host a server.

## Browser Viewer

Any device on the same local network can open:

```text
http://<pc-lan-ip>:5055/
```

For smart TVs, try:

```text
http://<pc-lan-ip>:5055/tv?fps=60&w=1280
```

## Compatibility Notes

- Hidden overscan works best with games that render to a real oversized GPU/window backbuffer.
- Some games or classic desktop apps may resize the window but fail to render useful content in the hidden area.
- AC EVO and iRacing workflows are included, but every game/GPU/display setup can behave differently.
- If a game disappears or is placed off-screen, use PANIC restore in the Windows app.
- If the crop is black, first verify the full preview and try a smaller bottom overscan height.

## Known Risk

Changing game window size or using custom display modes can cause games to reset, minimize, or crash. Do not use EDID/CRU changes unless you know how to recover your display.
