## Quick Start

1. Download and unzip `RealDashMirror-V1.0.5-Free-Exe.zip`.
2. Launch `DashMirrorOverscan.exe`.
3. Start the racing game in windowed or borderless mode.
4. Choose the game/source in the Assistant.
5. Apply the recommended bottom overscan.
6. Refresh the game resolution/placement if needed.
7. Place and validate the dashboard crop.
8. Launch the viewer and scan the QR code or open the LAN URL on Android, phone, tablet, or TV.

## Compatibility Notes

Hidden overscan depends on how each game renders an oversized window/backbuffer. Some games can resize but still refuse to render useful content in the hidden area. If the crop is black, inspect the full preview first, try a smaller bottom overscan height, and use PANIC restore if the game window is lost.
